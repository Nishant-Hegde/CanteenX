# CanteenX - Simple MERN Canteen App

## How to run

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Edit API endpoints in App.js if running backend elsewhere.

```bash
cd frontend
npm install
npm start
```

- Browse [http://localhost:3000](http://localhost:3000) for app.
- Menu, cart, order placement for users.
- Simple admin panel to add menu items & view orders.
